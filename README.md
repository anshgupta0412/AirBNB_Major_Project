# 🏡 Airbnb Clone (Major Project)

A full-stack(MERN) web application inspired by Airbnb, allowing users to explore, list, and manage rental properties. This project demonstrates real-world backend architecture with MVC pattern and dynamic frontend rendering.

## 🚀 Features
- 🔐 User Authentication (Login / Register)
- 🏠 Add, edit, and delete property listings
- 📍 Location-based property details
- 🖼️ Image upload for listings
- 📂 MVC architecture (Models, Views, Controllers)
- 📊 Organized backend with routes & utilities
- 💻 Responsive UI for better user experience

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- Templating: EJS

## 📁 Project Structure
- controllers/
- models/
- routes/
- views/
- public/
- utils/
- uploads/
